# Yummy 🍳

Yummy 是一款基于 iOS 的智能食谱管理应用，利用 AI 技术帮助用户生成、整理和管理个性化食谱。

## ✨ 主要功能

*   **AI 食谱生成**: 结合大模型能力，通过简单的描述即可自动生成包含食材、步骤、小贴士的完整食谱。
*   **食谱管理**: 支持食谱的查看、编辑、分类（标签）和本地持久化存储。
*   **料理清单**: 一键将食谱加入“料理清单”，自动拆分为采购、备菜、烹饪三个阶段，辅助烹饪全流程。
*   **精美分享**: 生成精美的食谱长图，方便分享给亲朋好友。
*   **沉浸式体验**: 采用 SwiftUI 构建的现代化 UI，支持触觉反馈和流畅的动画交互。

## 🛠 技术栈

*   **UI 框架**: SwiftUI
*   **架构模式**: MVVM
*   **数据存储**: CoreData
*   **网络请求**: URLSession
*   **AI 服务**: 集成大模型 API (BigModel)
*   **其他**: Combine, PhotosUI, UserNotifications

## 🚀 快速开始

1.  克隆仓库：
    ```bash
    git clone git@github.com:gym0915/yummy.git
    ```
2.  使用 Xcode 打开 `yummy.xcodeproj`。
3.  配置签名证书（Signing & Capabilities）。
4.  运行项目 (Cmd + R)。
